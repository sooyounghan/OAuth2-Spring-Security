-----
### ProviderSettings (AuthorizationServerSettings)
-----
1. OAuth2 인증 서버 (공급자)에 대한 구성 설정 포함
2. 프로토콜 엔드포인트에 대한 URI와 발급자의 식별자를 지정
3. 초기화 시 issuer를 제외하고는 기본 등록이 됨
